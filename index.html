Alumini Network App

     

I developed an alumni network application with social media-like feed features utilizing the MVVM (Model-View-ViewModel) architecture in both Kotlin and Java programming languages. This architecture separates the presentation logic from the business logic, making the app's development process more efficient and maintainable. The social media-like feed features allow users to easily view and interact with updates from other alumni, fostering a sense of community and engagement within the network. Overall, the app provides a modern and user-friendly platform for alumni to connect, communicate, and collaborate with each other.
